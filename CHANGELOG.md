# Changelog — AI Tag Suggestions

## 1.0

- Initial release.
- Suggest hierarchical tags for a photo using the Claude vision API.
- AI icon next to the tag input opens a review-and-apply suggestion panel.
- Per-photo analysis stored in the library; shown on an **AI Tags** Properties tab.
- Analysed photos marked with an **AI** badge in the gallery.
- API key configured in **Settings → AI Tags** (stored encrypted with Windows DPAPI).
